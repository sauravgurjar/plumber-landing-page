# NorthStar Plumbing (Vite + React + Tailwind)

Production-ready multi-page plumbing service website template for Canadian homeowners (35–50): clean, trustworthy, and mobile-first.

## Tech

- Vite + React
- Tailwind CSS
- React Router
- SEO via `react-helmet-async`
- Subtle animations via `framer-motion`
- Icons via `lucide-react`

## Getting Started

```bash
npm install
npm run dev
```

## Pages

- `/` Home
- `/services` Services
- `/reviews` Reviews
- `/contact` Contact

## Customize

- Update brand/contact info and content in `src/content/siteData.js`
- Replace map placeholder on `src/pages/ContactPage.jsx`
- Update meta title/description defaults in `index.html` and per-page via `src/components/SEO.jsx`
